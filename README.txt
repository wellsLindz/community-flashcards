Flashcards - Modern Version

How to use:
1. Keep flashcards.html in the same folder as the fronts and backs folders.
2. Double-click flashcards.html to open it in a web browser.
3. Click the card or press Space to flip.

Shortcuts:
Space = Flip
Left Arrow = Previous
Right Arrow = Next
R = Shuffle
Z = Zoom
Esc = Close zoom

To make a new deck later:
Put front images in the fronts folder and matching back images in the backs folder. Matching cards should have the same main filename after the words front/back.
